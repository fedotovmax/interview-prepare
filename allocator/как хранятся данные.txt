1. Массив
   - резервируется непрерывный блок памяти для всех его элементов (массив статический, размер и тип данных известен)
   - располагаются строго по порядку
   - массив != указатель, внутри него нет указателя. это "структура", у которой есть адрес начала, длина и размер элементов
   - формула доступа по индексу *(array_addr + index *element_size)
   - не имеет постоянного расположения, может переезжать, когда растет стек горутин

const element_size = unsafe.Sizeof(int32(0))
array := [...]int32{1,2,3}
array_addr := unsafe.Pointer(&array)

first := *(*int32)(unsafe.Add(array_addr, 0 * element_size))
